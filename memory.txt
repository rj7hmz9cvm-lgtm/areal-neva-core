
**Bash-команда:**  
```bash
tail -n 50 memory.txt > memory.tmp && mv memory.tmp memory.txt && systemctl restart orchestration.service
```  

**Пояснение:**  
1. `tail -n 50` сохраняет последние 50 строк `memory.txt` во временный файл.  
2. `mv` заменяет исходный файл временным.  
3. `systemctl restart` перезапускает сервис оркестра.  

Если `logs.bot.log` недоступен, требуется путь к файлу или дополнительные данные. Для поиска ошибок по IP:8080 в логах используйте:  
```bash
grep -E "ERROR.*8080" logs.bot.log | tail -n 20
```
