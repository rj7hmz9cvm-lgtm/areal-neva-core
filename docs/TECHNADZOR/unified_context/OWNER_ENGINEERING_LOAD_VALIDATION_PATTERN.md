## Паттерн

Владелец применяет сравнительную нагрузочную валидацию при оценке конструктивных решений:

```
600 кг/м²  — нормативная или типовая расчётная нагрузка (опорное значение)
1500 кг/м² — предельная или проверочная нагрузка (контрольное значение)
```

Логика: если конструктив выдерживает 1500 кг/м² при норме 600 кг/м² — запас прочности подтверждён.
