## Профиль — каркасный дом

Типовые узлы из актов:
- окна (установка, монтажный шов)
- стойки каркаса
- OSB обшивка
- черновой пол / лаги
- напольный зазор / продухи
- основание под полусухую стяжку

Базовые нормы:
- СП 70 — несущие конструкции
- ГОСТ 30971 — если окна: **статус NOT_PRESENT_IN_NORMATIVE_ENGINE — кандидат для будущего добавления**

Примечание по ГОСТ 30971:
```
Не добавлен в normative_engine.py.
Если нужна логика монтажного шва:
  писать "норма по монтажному шву требует отдельного добавления/проверки"
Не изобретать пункты.
```
