def parse_khl_boxscore(url):
    html = requests.get(url).text
    soup = BeautifulSoup(html, 'html.parser')

**2.2. Ключевые метрики (формулы)**
- **Corsi (CF%)** = (Броски команды) / (Броски команды + Броски соперника) * 100  
  `где броски = SOG + missed shots + blocked shots`
- **xG (ожидаемые голы)** = Σ (вероятность гола для каждого броска)  
  `используются веса из моделей MoneyPuck или NaturalStatTrick`
- **PDO** = On-ice Sh% + On-ice Sv%  
  `нормальное значение ≈100, отклонения указывают на удачу/неудачу`

**2.3. Пример расчёта в Python**
```python
def calculate_metrics(team_data):
    corsi_for = team_data['SOG'] + team_data['Missed'] + team_data['Blocked']
    corsi_against = team_data['SOG_against'] + team_data['Missed_against'] + team_data['Blocked_against']
    cf_pct = corsi_for / (corsi_for + corsi_against) * 100
    pdo = (team_data['Goals'] / team_data['SOG'] * 100) + (1 - team_data['Goals_against'] / team_data['SOG_against']) * 100
    return {'CF%': cf_pct, 'PDO': pdo}
```

#### **4. Сравнение с букмекерами**
**4.1. Value Betting:**
- Если модель даёт вероятность 55%, а букмекер предлагает коэффициент 2.0 (подразумеваемая вероятность 50%):
  - **Value** = (55% * 2.0) - 1 = 0.1 (10% ожидаемой прибыли).

**4.2. Автоматизация поиска Value:**
```python
def find_value_bets(model, games_today):
    value_bets = []
    for game in games_today:
        prob = model.predict_proba([game['features']])[0][1]
        implied_prob = 1 / game['odds']
        if prob > implied_prob + 0.05:  # Порог value
            value_bets.append(game)
    return value_bets
```

#### **5. Рекомендации по ставкам**
**5.1. Стратегии:**
- **Live-ставки**:  
  - Если команда с CF% >55% проигрывает после 2-го периода, ставка на её победу/ничью в основное время.
- **Тоталы**:  
  - При xG >3.5 за игру и коэффициенте на ТБ 2.5 >1.9 — ставка на "больше".

**5.2. Управление банком:**
- **Критерий Келли**:  
  `Размер ставки = (вероятность * коэффициент - 1) / (коэффициент - 1)`
