# ORCHESTRA DOMAIN CAPABILITIES

## DOCS / FILE OPS
- PDF: читать, собирать, разбивать, объединять, извлекать текст, создавать
- DOCX: создавать, править, собирать отчеты, акты, заключения
- XLSX: сметы, расчеты, формулы, правки без поломки формул
- PPTX: собирать презентации
- изображения: сортировка, переименование, упаковка, подборки
- архивы: zip/unzip, раскладка по структуре проекта

## CAD / DRAWINGS
- DWG/DXF: разбор, конвертация, подготовка к чтению
- извлечение данных из чертежей
- DWG → DXF → PDF/таблицы/текст
- работа с листами, спецификациями, ведомостями

## TEHNADZOR
- приемка квартир и домов
- замечания
- дефектовки
- акты
- заключения
- фотофиксация
- сводки по нарушениям
- материалы для суда
- разбор исполнительной и проектной документации

## PROJECT / CONSTRUCTION
- разбор АР / КЖ / КМ / инженерки
- объемы
- ведомости
- расчеты материалов
- сравнение версий проекта
- извлечение позиций из PDF / DWG / таблиц

## ESTIMATES / TABLES
- сметы
- калькуляции
- формулы
- сверка объемов
- перенос в Excel / Google Sheets
- приведение таблиц в рабочий вид

## TELEGRAM WORKFLOW
- принимать задачи из Telegram
- раскладывать файлы по проектным папкам
- собирать переписки в текст
- отдельно деньги / документы / фото / переписки
- отдавать готовый результат

## SEARCH / ANALYTICS
- поиск данных по объекту
- поиск норм и техинформации
- сравнение решений
- аналитика по стройке, материалам, инструментам, подрядчикам

## OUTPUTS
- готовый файл
- готовая таблица
- готовый текст
- готовый отчет
- готовый архив проекта
- без фейковых заглушек
